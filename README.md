# ポケストポスカ確認用 (PokeStop Postcard Viewer)

ポケストップのフォトディスク、およびギフト（ポストカード）として表示された際の視覚的なイメージを、ブラウザ上でシミュレートするためのWebツールです。

## 概要
手持ちの画像を読み込ませることで、円形のディスク形式と正方形のカード形式でのトリミングや見栄えを即座に確認できます。画像処理はすべてブラウザ内で行われるため、外部サーバーへのデータ送信は発生しません。

## 機能
- 表示モードの切り替え（ディスク / カード）
- 画像のドラッグ&ドロップによる読み込み
- 複数画像のプレビューおよび切り替え（矢印ボタン、キーボードの左右キー、スワイプ操作）
- サムネイル一覧からの画像選択

## 使い方
1. index.html をブラウザで開きます。
2. 画像ファイルを画面上にドロップするか、下部の「画像を追加」から選択します。
3. 上部のボタンで「ディスク」と「カード」の各モードを切り替えて確認してください。
